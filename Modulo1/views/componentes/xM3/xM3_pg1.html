Transferencia de Material desde el Acopio

Control del Colector de Polvo

El operador desde la sala de control puede accionar el colector de polvo y el ventilador de presurización del túnel para mantener las condiciones ambientales adecuadas.

Función clave: Accionamiento manual desde sala de control

Factor de Segregación

El operador puede ingresar manualmente el factor de segregación en el sistema de control para ajustar la mezcla de material en las fajas alimentadoras.

Control: Manual
Ubicación: Sistema de control

Válvula Rotatoria

La válvula rotatoria del colector de polvo puede activarse manualmente según el nivel en la tolva, permitiendo un control preciso del flujo de material.

Criterio de activación: Nivel en la tolva

Molienda SAG

Control de Velocidad

El operador en sala de control puede ajustar manualmente la referencia de velocidad del controlador de flujo másico del molino SAG para optimizar la operación.

Parámetro: Velocidad de referencia
Tipo: Flujo másico

Sentido de Giro

El operador puede seleccionar el sentido de giro del molino según los requerimientos operacionales específicos del proceso.

Monitoreo de Variables

Control continuo de variables críticas para mantener la operación dentro de parámetros seguros:

Velocidad de rotación
Carga
Potencia
Presión en descansos

Gestión de Alarmas

Si se presentan alarmas, el operador debe verificar y ajustar parámetros operativos para mantener dentro de los rangos seguros de operación.

Protocolo: Verificación y ajuste de parámetros operativos
